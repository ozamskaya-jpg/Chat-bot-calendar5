# 🤖 Чат-бот для записи клиентов

Telegram-бот для онлайн-записи клиентов с уведомлениями и HTML-календарём.

## Возможности
- 📅 Записаться на приём
- 🔄 Перенести запись
- 📋 Просмотреть свои записи
- ❌ Отменить запись
- 🔔 Напоминания за 24 часа и за 1 час

## Переменные окружения (Railway → Variables)

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Токен бота от @BotFather |
| `OWNER_ID` | Ваш Telegram ID (узнать у @userinfobot) |
| `DATABASE_URL` | Строка подключения PostgreSQL |

## Команды для владельца
- `/day` — записи на сегодня
- `/week` — записи на неделю
- `/calendar` — HTML-файл с расписанием
- `/block ГГГГ-ММ-ДД ЧЧ:ММ [название]` — заблокировать время
- `/export` — выгрузить базу в JSON
